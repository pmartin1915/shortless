## Full Description (Chrome Web Store — plain text)

Shortless surgically removes short-form video content from your browsing experience while leaving everything else untouched. Unlike full domain blockers, Shortless targets only the short-form feeds and players — you keep full access to YouTube videos, Instagram profiles, and Snapchat stories.

SUPPORTED PLATFORMS
- YouTube Shorts (blocked and redirected to regular video player)
- Instagram Reels (hidden from feed, Explore, and profile tabs)
- TikTok (full domain block — TikTok is short-form only)
- Snapchat Spotlight (hidden from Snapchat's web experience)

HOW IT WORKS
Shortless uses a 3-layer defense system to ensure nothing slips through:

1. Network Rules — Chrome's declarativeNetRequest API blocks or redirects short-form requests before they even load, saving bandwidth and preventing flicker.
2. CSS Injection — Stylesheets injected at page load instantly hide short-form UI elements so you never see them flash on screen.
3. Content Scripts — A MutationObserver watches for dynamically loaded short-form content (common in single-page apps) and hides it in real time.

PRIVACY FOCUSED
- Zero data collection — no analytics, no tracking, no cookies
- No external network requests — everything runs on your device
- No user accounts or sign-up required
- Open source filter rules you can inspect yourself
- Uses Chrome's built-in sync only to remember your preferences

FEATURES
- Per-platform toggles — disable blocking for any individual platform from the popup
- Block counter — see how many short-form items have been blocked today
- Lightweight — minimal performance impact, no background processes
- Dark mode popup UI

Free, open source, and private by design. Take back control of your attention.
